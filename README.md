# tpl-gulpfile


## Description
gulpfileのテンプレートです。  
今まで、pugとphpを分けて管理し、その後各テンプレートに入れこむやり方をしていましたが、  
pugとphpのgulpfileを一緒にして管理することにしました。


## node
v14.14.0


## gulp
v4.0.2



## Command

### init
```
npm i
```

### dafault
```
gulp
```

### watch
```
gulp watch
```


それぞれのタスクを確認したい場合は、gulpfile.jsの中身を見てください。
